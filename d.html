Polynomial Constant 'c' Finder

This tool calculates the constant term 'c' of a polynomial equation given its roots in various bases. Input the roots as JSON data, and the application will compute the constant using the product of roots method.

Result:
Enter JSON and click calculate to see the result
Example JSON Format:
{ "keys": { "n": 4, "k": 3 }, "1": { "base": "10", "value": "4" }, "2": { "base": "2", "value": "111" }, "3": { "base": "10", "value": "12" }, "6": { "base": "4", "value": "213" } }
